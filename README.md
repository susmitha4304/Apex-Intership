# Apex-Intership
# Web Development Tasks

This repository contains a collection of web development projects created using **HTML**, **CSS**, and **JavaScript**. Each task focuses on different frontend development concepts.

##  Projects Included

### Task 1: To-Do List Application
A simple and interactive task management application that allows users to:
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Manage daily activities efficiently

**Technologies Used:** HTML, CSS, JavaScript

---

### Task 2: Interactive Quiz Platform
A dynamic quiz application designed to test user knowledge through multiple-choice questions.

**Features:**
- Multiple-choice questions
- Score calculation
- Instant feedback
- Interactive user interface

**Technologies Used:** HTML, CSS, JavaScript

---

### Task 3: Simple Product Listing E-Commerce Page
A basic e-commerce product listing page showcasing products in a clean and responsive layout.

**Features:**
- Product cards with images and descriptions
- Responsive design
- Modern UI styling
- Organized product display

**Technologies Used:** HTML, CSS, JavaScript

---

### Task 4: Frontend Portfolio Website
A professional portfolio website highlighting frontend development skills and projects.

**Features:**
- About section
- Skills showcase
- Projects section
- Contact information
- Responsive design

**Technologies Used:** HTML, CSS, JavaScript

---

### Task 5: Personal Portfolio Website
A personal portfolio website created to represent personal achievements, skills, and career goals.

**Features:**
- Personal introduction

- Project showcase

- Responsive layout

**Technologies Used:** HTML, CSS, JavaScript
